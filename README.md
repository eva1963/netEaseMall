# netEaseMall
> 本项目是模仿网易严选wrap端，基于react来搭建的移动端，里面主要使用redux/react-redux/react-router-dom以及中间件redux-logger/redux-promise等完成

为了防止跟您的其他项目端口冲突，我已将默认端口需改为63342；

开发项目，需要先启动server后台数据支持，注意！！

先进入code，进行初始化本项目：

- `yarn install`

启动项目：

- `yarn start`

生成开发环境包：

- `yarn build`
